Założenia:\\
\begin{center}
	$ x \ge 0$  i  $\log_{[z]} x \ge 0 $\\
	$ x \ge 0$  i  $x \ge [z]^{0} $\\
	$ x \ge 0$  i  $ x \ge [p] $\\
\end{center}
Rozwiązanie:
\begin{center}
	$ \log_{[z]} x \le (\frac{1}{3})^{0} $\\
	$ \log_{[z]} x \le [u] \Rightarrow x \le [z]^{[u]} \Rightarrow x \le [w] $\\
\end{center}
$x \le [w]$ oraz założenie, że $ x \ge [p] $\\
Otrzymujemy $ x \in ([p],[w]> $
Koniec rozwiązania.\\
